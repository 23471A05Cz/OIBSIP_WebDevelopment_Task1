# OIBSIP Web Development Internship – Level 2

This repository contains the projects completed as part of the **Oasis Infobyte Web Development Internship – Level 2**.

## Task 1: Calculator

### Project Title
Simple Web Calculator

### Objective
The objective of this project is to create a simple web-based calculator that performs basic arithmetic operations using HTML, CSS, and JavaScript.

### Technologies Used
- HTML
- CSS
- JavaScript

### Description
This project is a user-friendly calculator that can perform addition, subtraction, multiplication, and division.  
The interface is simple and responsive for easy use.

### Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear display function
